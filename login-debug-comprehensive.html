📋 Hướng dẫn khắc phục

Các bước khắc phục phổ biến: 1. 🔍 Kiểm tra trạng thái user - Xác định user có tồn tại trong database không - Kiểm tra email đã được xác thực chưa 2. 📧 Nếu chưa xác thực email - Sử dụng nút "Gửi lại email xác thực" - Kiểm tra hộp thư spam/junk - Nhập mã xác thực đã nhận 3. ❌ Nếu user không tồn tại - Đăng ký lại tài khoản - Hoàn tất xác thực email 4. 🔐 Nếu mật khẩu sai - Kiểm tra lại mật khẩu đã nhập - Sử dụng tính năng quên mật khẩu (nếu có) 5. 🔗 Kiểm tra kết nối Backend - Đảm bảo server đang chạy - Kiểm tra API endpoints hoạt động
